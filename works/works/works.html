PROJECT GOAL

HLB 제약은 HLB KOREA 계열 제약 회사입니다. 일반 의약품, 전문 의약품을 포함하여 건강기능식품과 희귀질환 관련 치료제의 개발, 임상 실험을 진행하고 있습니다.

본 프로젝트에서는 제약회사가 주는 전문적인 이미지를 최대한 간결하게 나타내려 하였으며 화면은 크롬 브라우저를 기준으로 제작되었습니다.

· 제약회사의 안정적이고 전문적인 느낌을 메인 화면에 제시.

· 회사의 로고 색상(주황색)으로 마우스 오버 색상 지정

· 주황색의 다소 가벼운 느낌에 남색으로 균형을 맞춤

· 배경 이미지가 있는 섹션에서 배경 이미지를 크게 방해하지 않으면서 하얀색 글자의 가독성을 높이기 위해 남색에 불투명도를 부여하여 텍스트 부분의 배경색 적용.

· 전체 메뉴를 보는 항목은 불필요하므로 삭제.

· 각 섹션의 움직임은 부드러움을 강조하기 위해 여유있게 1초로 설정.

· News&Notice 부분의 슬라이드는 화살표를 누르면 무한으로 반복. 다음 컨텐츠가 있음을 사용자게에 쉽게 안내하기 위하여 한 번에 3.5개의 슬라이드가 보이게 제작.

· 각 섹션에 해당하는 부분은 창 양 끝에 숨겨있다가 해당 스크롤에 도달하면 자연스럽게 등장. aos.js 파일 사용.

· 섹션 제목 왼쪽에 붙는 긴 선은 css의 padding 값과 border 값으로 제어.

· 대부분의 정렬에 flex 사용.

· 웹 브라우저에 맞춰 제작하였으므로 body에는 min-width값으로 1200px 적용.

HBL 제약 웹 페이지 제작은 제가 처음으로 디자인과 퍼블리싱을 온전히 자력으로 해낸 프로젝트입니다.

웹 화면을 기준으로 제작을 하다보니 높이의 값을 px로 주는 것이 좋을지, vh로 주는 것이 좋을지 고민을 많이 했습니다. 결론적으로는 px값으로 처리하였고, 나름대로 만족할만한 결과가 되었다고 생각합니다.

aos 스크립트를 적용할 때 New&Notice 부분이 나타나기 전에 오른쪽으로 스크롤이 생기는 문제가 있었습니다.이는 section의 부모요소인 main에 width 값 100%를 부여하고, overflow 를 hidden으로 처리하여 온전하게 만들었습니다.

모쪼록 해당 프로젝트로 보는 이로 편리함을 느끼게 하고, 디자인적으로 아름답게 보이기를 바랍니다.
